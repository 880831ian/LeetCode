## 9 _ 回文數量

給一個整數 `x` ，如果 `x` 回文跟原先的一樣，就返回 `true` ，否則返回 `false` 。

當一個整數從後面讀回來與前面讀一樣的話，他就是回文。

* 例如 121 是回文， 123 不是。

**範例 1:**

```
輸入: x = 121
輸出: true
解釋: 121 從左到右和從右到左讀都是 121。
```

<br>

**範例 2:**

```
輸入: x = -121
輸出: false
解釋: 從左讀到右是 -121 。從右讀到左是 121- 。因此它不是回文。
```

<br>

**範例 3:**

```
輸入: x = 10
輸出: false
解釋: 從右邊讀取是 01，所以它不是回文。
```

<br>

限制:

* -2^31 <= x <= 2^31 - 1
