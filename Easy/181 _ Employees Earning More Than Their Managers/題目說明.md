## 員工收入超過經理

Table: `Employee`

```
+-------------+---------+
| Column Name | Type    |
+-------------+---------+
| id          | int     |
| name        | varchar |
| salary      | int     |
| managerId   | int     |
+-------------+---------+
id 是該表的主 key
此表的每一行都表示員工的 ID、姓名、薪水、和經理的 ID。
```

<br>

編寫 SQL 查詢以查找收入高於其經理的員工。

以**任意順序**返回結果表。

查詢解果格式如下例所示。
<br>

**範例 1:**

```
輸入: 
Employee table:
+----+-------+--------+-----------+
| id | name  | salary | managerId |
+----+-------+--------+-----------+
| 1  | Joe   | 70000  | 3         |
| 2  | Henry | 80000  | 4         |
| 3  | Sam   | 60000  | Null      |
| 4  | Max   | 90000  | Null      |
+----+-------+--------+-----------+
輸出: 
+----------+
| Employee |
+----------+
| Joe      |
+----------+
解釋: Joe 是唯一一個收入超過他的經理的員工。
```
